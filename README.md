# 📸 AK Studio Digital Album Website

This is a premium photography website with a digital album feature.

## 🚀 Features
- Cinematic design (black + gold theme)
- WhatsApp booking system
- Digital album (client-wise)
- Password protected album
- Responsive design (mobile + desktop)

## 📁 Album System
- Albums are stored inside `/albums/` folder
- Each client has a separate folder
- Example:
  albums/rahul-wedding/page1.jpg

## 🔐 Album Access
- Password protected album
- Example password: AKStudio@2026
